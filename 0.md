#### **2. 为什么英语很重要？**

编程世界的第一官方语言是**英语**。所有最新的技术、最权威的官方文档、最活跃的开发者社区、最经典的编程书籍，几乎都是英文的。拥有基础的英语阅读能力，意味着你能直接获取第一手信息，而不是等待别人翻译。**别担心，你不需要精通，只需要敢于阅读，借助翻译工具，慢慢就能适应。**

#### **3. 必备的软件与工具 (Windows)**

这些是我们接下来每天都要打交道的“神兵利器”，请务必安装并熟悉它们。

*   **浏览器 - Google Chrome**
    *   **它是什么**：不仅是上网工具，更是前端开发者的“第一神器”。
    *   **安装**：访问 [Google Chrome官网](https://www.google.com/chrome/) 下载安装。
    *   **核心技能**：务必学会按 `F12` 打开开发者工具，重点关注 `Elements` (看页面骨架), `Console` (看代码报错), `Network` (看网络通信) 这三个面板。

*   **代码编辑器 - Visual Studio Code (VS Code)**
    *   **它是什么**：我们写代码的地方，目前全球最流行、免费又强大的编辑器。
    *   **安装**：访问 [VS Code官网](https://code.visualstudio.com/) 下载安装。
    *   **必装插件** (打开VS Code，在左侧扩展商店搜索安装)：
        *   `Chinese (Simplified) Language Pack`: 中文汉化包。
        *   `Live Server`: 实时预览网页，你一保存代码，网页就自动刷新。
        *   `Prettier - Code formatter`: 代码“美容师”，一键让你的代码变得整齐漂亮。
        *   `Auto Rename Tag`: 自动修改匹配的HTML标签，省时省力。

*   **代码时光机 - Git & Gitee**
    *   **它是什么**：Git是一个“版本控制”工具，就像游戏的存档系统。它能帮你记录每一次代码的修改，让你随时可以“读档”回到之前的任何一个版本。
    *   **为什么需要**：这是程序员协作和管理代码的必备技能。
    *   **考虑到网络情况，我们使用Gitee**：GitHub是全球最大的代码托管平台，但国内访问可能不稳定。因此，我建议大家使用[Gitee (码云)](https://gitee.com/)，它是中国的类似平台，访问速度快，功能同样强大。
    *   **安装**：访问 [Git官网](https://git-scm.com/downloads) 下载安装，安装时一路点击“Next”即可。
    *   **核心技能**：学会打开Git Bash（一个命令行窗口），练习 `git add` (把文件放进暂存区), `git commit` (生成一个存档), `git push` (把存档推送到Gitee) 这几个核心命令。
    *   **教程**：[廖雪峰的Git教程](https://www.liaoxuefeng.com/wiki/896043488029600) 是非常棒的中文入门教程。

### **第二部分：前端基础：亲手创造你的第一个网页 (第2周 - 第8周)**

**本周目标**：掌握前端三剑客，能够独立制作出功能丰富的静态网页。

#### **什么是前端开发？(一些术语澄清)**

很简单，你现在浏览器里看到的这个页面，就是“前端”。前端开发就是创造用户能直接看到和交互的界面的工作。我们常说的“前端三剑客”是：

*   **HTML (超文本标记语言)**：网页的**骨架**。它决定了网页上有什么内容，比如标题、段落、图片、按钮。
*   **CSS (层叠样式表)**：网页的**皮肤**和**服装**。它负责美化网页，控制内容的颜色、大小、位置和布局。
*   **JavaScript (JS)**：网页的**灵魂**和**肌肉**。它让网页“活”起来，负责实现交互功能，比如点击按钮后的弹窗、网页上的轮播图等。

#### **阶段一：HTML - 搭建网页的骨架 (2周)**
*   **学习内容**：常用标签 (`<h1>`, `<p>`, `<a>`, `<img>`)，列表，表格，表单，以及最重要的**语义化标签** (`<header>`, `<nav>`, `<main>`, `<footer>`)。
*   **权威资源**：
    *   **主线任务**：[freeCodeCamp](https://www.freecodecamp.org/learn) 的 "Responsive Web Design" 认证课程，从零开始，交互式学习，强烈推荐！
    *   **中文视频**：B站搜索“黑马程序员前端HTML”或“尚硅谷前端HTML”。
    *   **权威字典**：[MDN Web Docs (HTML)](https://developer.mozilla.org/zh-CN/docs/Web/HTML)，遇到不确定的标签或属性，查它！

#### **阶段二：CSS - 美化你的网页 (3周)**
*   **学习内容**：选择器，盒子模型（**核心中的核心**），布局技术（**Flexbox** 和 **Grid** 是现代网页布局的基石，必须掌握），以及过渡、动画等特效。
*   **权威资源**：
    *   **主线任务**：继续 freeCodeCamp 的课程。
    *   **游戏化练习**：
        *   [Flexbox Froggy](https://flexboxfroggy.com/): 通过把青蛙放到荷叶上，彻底学会Flexbox。
        *   [Grid Garden](https://cssgridgarden.com/): 用CSS网格浇灌你的花园。
    *   **权威字典**：[MDN Web Docs (CSS)](https://developer.mozilla.org/zh-CN/docs/Web/CSS)。

#### **阶段三：JavaScript - 赋予网页生命 (3周)**
*   **学习内容**：基础语法（变量、数据类型、循环、函数等），**DOM操作**（用JS控制HTML元素，这是前端交互的核心），**事件处理**（响应用户的点击、滚动等行为）。
*   **权威资源**：
    *   **主线任务**：freeCodeCamp 的 "JavaScript Algorithms and Data Structures" 认证课程。
    *   **在线教程**：[现代 JavaScript 教程](https://zh.javascript.info/)，内容详实且现代。
    *   **免费书籍**：在 [free-programming-books](https://github.com/EbookFoundation/free-programming-books) 中查找《Eloquent JavaScript》等经典免费书籍。
    *   **权威字典**：[MDN Web Docs (JavaScript)](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript)。

当你掌握了基础三剑客，恭喜你，已经入门了！接下来，我们要学习使用更现代化的工具，提升开发效率。

*   **前端框架入门 - Vue.js (推荐)**
    *   **它是什么**：一个能让你更高效、更结构化地开发复杂页面的“脚手架”。Vue以其易学性著称，对新手非常友好。
    *   **权威资源**：[Vue.js官方文档](https://cn.vuejs.org/) 就是最好的教程，写得像书一样清晰易懂。
    *   **下一步**：学习了Vue之后，可以继续探索 `React` 框架，或者学习 `Node.js` 让自己具备后端开发能力，向“全栈工程师”迈进。

### **第四部分：激励、协作与持续进步**

一个人走得快，一群人走得远。为了让大家保持学习的热情，互相鼓励，我设计了以下机制：

1.  **每周分享会 (Show & Tell)**：每周固定时间，每个人分享自己本周的学习成果或踩过的坑，哪怕只是一个小作品。这是你展示自己、获得反馈的舞台。
2.  **代码审查 (Code Review)**：我们将互相查看代码（通过Gitee链接），提出建设性的改进建议。这是学习他人优点、习惯团队开发模式的绝佳机会。
3.  **学习小组 (Study Group)**：后面会带大家共同完成一个稍大的项目，比如合作仿写一个完整的网站。
4.  **Gitee贡献图 (点亮绿墙)**：我鼓励大家每天都进行代码提交，让你的Gitee贡献图变得“绿意盎然”。这是一种非常直观的“打卡”方式，能带来持续的成就感。
5.  **每日答疑 (Q&A)**：**学习中遇到任何问题，不要憋着！随时可以在我们的群里提出来。我看到后会第一时间为你们解答。记住，提问是学习的重要一环。**

最后，我想说，计算机科学是一条需要终生学习的道路。我很高兴能在这段旅程的开端陪伴你们，让我们一起学习，一起进步！

出发吧，未来的创造者们！
