## 🚀 快速开始

### 5 分钟体验 AI 工具

1. **安装 Open-Claude-Cowork**
   - 访问：https://github.com/DevAgentForge/Open-Claude-Cowork
   - 下载 macOS/Windows 安装包
   - 安装并启动

2. **配置 AI 模型**
   - 注册智谱 GLM：https://open.bigmodel.cn
   - 获取 API Key
   - 在 Open-Cowork 中配置

3. **尝试第一个任务**
   - 准备一个 PDF 文件
   - 参考 [英语绘本学习助手](AI-Zero-to-Hero/english-learning-guide.md)
   - 复制任务模板，开始使用
