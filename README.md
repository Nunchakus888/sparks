# Sparks - AI 学习资源导航

## 📚 项目简介

本项目收集整理了 AI 工具学习资源，专注于帮助**非技术人员**从零开始掌握 AI 工具的使用。

---

## 🏁 最终目标

学完本项目，你将能够：

- **独立使用 AI 工具**处理日常工作（文档整理、数据分析、内容生成）
- **用自然语言描述任务**，让 AI 自动完成（不需要写代码）
- **设计自己的工作流**，把重复性任务交给 AI（如英语学习、文件整理、报告生成）
- **节省 50%+ 的时间**在文档处理和内容创作上

---

## 📋 学习路径（按顺序进行）

> **学习方法**：每个阶段先通读文档，把不懂的问题记下来，带着问题进入下一步实操。
> 遇到卡住的地方不要死磕，先标记，继续往后走，很多问题在实操中会自然解决。

### 阶段 1：了解全貌（30 分钟）

**目标**：知道 AI 工具能做什么，建立整体认知

**阅读**：[AI 零基础到精通学习路径](AI-Zero-to-Hero/zero-to-hero.md)

**做什么**：
- [ ] 通读文档，了解学习路线
- [ ] 记下你最感兴趣的应用场景
- [ ] 记下看不懂的术语和概念

**完成标志**：能用一句话说出 "AI 工具可以帮我做什么"

---

### 阶段 2：安装工具（1-2 小时）

**目标**：完成工具安装和 API 配置，跑通第一个任务

**阅读**：[Open-Claude-Cowork 完整教程](AI-Zero-to-Hero/open-cowork-tutorial.md)

**做什么**：
- [ ] 安装 Open-Claude-Cowork
- [ ] 注册 AI 模型账号（推荐智谱 GLM），获取 API Key
- [ ] 配置并测试连接
- [ ] 完成教程中的"第一个任务"（创建待办清单）
- [ ] 记下安装过程中遇到的问题

**完成标志**：成功让 AI 完成一个简单任务（如创建一个文件）

---

### 阶段 3：实战上手（1-2 天）

**目标**：用真实场景练手，掌握"如何向 AI 描述任务"

**阅读**：[英语绘本学习助手](AI-Zero-to-Hero/english-learning-guide.md)

**做什么**：
- [ ] 准备一本英语绘本 PDF
- [ ] 复制文档中的任务模板，修改书名
- [ ] 让 AI 生成完整学习课程包
- [ ] 检查生成的词汇表、音频、练习题
- [ ] 尝试修改任务描述（换难度、加需求），观察输出变化
- [ ] 记下"什么样的描述效果好，什么样的效果差"

**完成标志**：独立完成一本绘本的课程包生成，并能调整任务描述获得不同结果

---

### 阶段 4：举一反三（1 周）

**目标**：把学到的方法应用到自己的实际工作中

**做什么**：
- [ ] 想 3 个你日常工作中重复性高的任务
- [ ] 参考英语学习的任务模板，写出你自己的任务描述
- [ ] 让 AI 执行，根据结果调整描述
- [ ] 把好用的任务描述保存为模板，以后复用

**举例**：
- 家长：批量生成不同绘本的学习材料
- 教师：从教材 PDF 提取知识点，生成课堂练习
- 职场人：整理会议录音文本，生成会议纪要
- 自由职业：批量翻译和润色文档

**完成标志**：有 2-3 个属于自己的任务模板，能持续复用

### 🗒️ 问题记录模板

学习过程中，用这个格式记录问题：

```
## 我的问题清单

### 阶段 1
- [ ] 问题：XXX 是什么意思？
- [ ] 问题：为什么要用 API Key？

### 阶段 2
- [ ] 问题：安装时报错 XXX，怎么解决？
- [ ] 问题：API Key 在哪里获取？

### 阶段 3
- [ ] 问题：生成的词汇表不够准确，怎么调整？
- [ ] 问题：能不能只生成音频不生成练习题？
```

> 带着问题清单来提问，能更快得到帮助。
> 很多问题在后续阶段会自然解决，先标记，不要卡住。
