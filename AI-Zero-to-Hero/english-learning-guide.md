# 英语绘本学习助手 - 完整使用指南

## 🎯 这是什么？

一个帮你从英语绘本 PDF 自动生成完整学习材料的 AI 助手，包括：
- 📚 词汇表（音标、释义、例句、用法）
- 🗣️ 语音朗读（单词、句子、全文）
- 📝 练习题（选择题、填空、翻译）
- 🎓 完整课程包（可直接使用）

**适合人群**：家长、教师、英语学习者（无需编程知识）

## 🚀 快速开始（5 分钟）

### Step 1: 安装 OpenWork

1. 访问：https://github.com/different-ai/openwork/releases
2. 下载最新版 `.dmg` 安装包
3. 安装并启动应用

### Step 2: 配置 AI 模型

1. 注册智谱 GLM（推荐）：https://open.bigmodel.cn
2. 获取 API Key
3. 在 OpenWork 中配置：
   - 点击"设置" → "模型配置"
   - 选择"智谱 GLM"
   - 输入 API Key
   - 选择模型：`glm-4-plus`
   - 测试连接

### Step 3: 准备 PDF 文件

将绘本 PDF 放入工作目录：
```
~/AI-Workspace/
└── books/
    └── the-very-hungry-caterpillar.pdf
```

### Step 4: 使用 AI 助手

在 OpenWork 中输入以下任务：

```
请帮我从 books/the-very-hungry-caterpillar.pdf 生成英语学习课程包：

【任务】
1. 提取 PDF 文本内容
2. 分析并提取：
   - 核心词汇（20-30个）：音标、词性、释义、例句、用法说明
   - 重要短语（10-15个）：含义、用法、例句
   - 典型句子（10-15个）：翻译、语法点分析
3. 为每个单词和句子生成英语朗读音频（使用免费的 gTTS）
4. 生成练习题：
   - 词汇测试（选择题 10 道）
   - 填空练习（10 道）
   - 翻译练习（5 道）
5. 将所有材料整理到 english-courses/the-very-hungry-caterpillar/ 目录

【要求】
- 目标难度：初级（A1）
- 输出格式：Markdown + HTML + 音频文件
- 生成 README.md 说明文档
- 创建 index.html 网页版课程

【输出结构】
english-courses/the-very-hungry-caterpillar/
├── README.md                    # 课程说明
├── materials/
│   ├── vocabulary.md            # 词汇表
│   ├── phrases.md               # 短语解析
│   ├── sentences.md             # 句子分析
│   └── full-text.md             # 完整文本
├── audio/
│   ├── vocabulary/              # 单词发音
│   ├── sentences/               # 句子朗读
│   └── full-reading.mp3         # 全文朗读
├── exercises/
│   ├── vocabulary-quiz.md       # 词汇测试
│   ├── fill-in-blanks.md        # 填空练习
│   └── translation.md           # 翻译练习
└── index.html                   # 网页版课程
```

### Step 5: 等待生成

AI 会自动完成所有步骤（约 3-5 分钟）：
```
📖 [1/5] 读取 PDF 文件...
   ✅ 共 15 页，识别到 342 个单词

🧠 [2/5] AI 智能分析...
   ✅ 提取关键词汇: 28 个
   ✅ 识别重要短语: 15 个
   ✅ 标记核心句子: 12 个

📝 [3/5] 生成学习材料...
   ✅ 词汇表、短语解析、句子分析、练习题

🔊 [4/5] 生成语音文件...
   ✅ 41 个音频文件

📦 [5/5] 打包课程...
   ✅ 课程包已生成！
```

### Step 6: 使用课程包

打开生成的 `index.html` 文件，即可在浏览器中学习。

## ❓ 常见问题

### Q1: PDF 提取不准确怎么办？

**问题**：扫描版 PDF 无法提取文字

**解决方案**：
```
在任务描述中添加：
【特殊说明】
- 这是扫描版 PDF，请使用 OCR 识别文字
- 或者：我会手动提供文本内容
```

然后将文本复制到 `.txt` 文件，让 AI 直接分析文本。

---

### Q2: 语音质量不满意？

**免费方案（gTTS）**：
- 质量：基础，机器感较强
- 适合：入门学习

**改进方案**：
在任务描述中指定：
```
【语音要求】
- 使用 OpenAI TTS（高质量，需要 OpenAI API Key）
- 声音：alloy（女声）或 onyx（男声）
- 语速：慢速（适合跟读）
```

---

### Q3: 如何批量处理多本绘本？

**方法**：
```
请批量处理 books 文件夹中的所有 PDF：
- 每本生成独立课程包
- 统一难度：初级（A1）
- 生成汇总目录（包含所有绘本的索引）
- 输出到 english-courses/ 目录
```

---

### Q4: 能否导出为 PDF 格式？

**可以！** 在任务描述中添加：
```
【输出格式】
- Markdown（原始格式）
- HTML（网页版）
- PDF（可打印版，使用 wkhtmltopdf 转换）
```

---

### Q5: 如何调整难度？

**难度等级**：
- **入门级（A1）**：3-8 岁儿童，基础词汇
- **初级（A2）**：小学生，简单句型
- **中级（B1）**：初中生/成人，复杂语法
- **高级（B2+）**：高中生/成人，文学分析

在任务描述中指定：
```
【难度】：中级（B1）
```
