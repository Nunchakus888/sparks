# AI Zero to Hero - 非技术人员实战路径

## 目标人群
- **AI 零基础**，无编程经验
- 需要用 AI 解决日常工作问题（文档处理、数据分析、自动化任务）
- 拥有 macOS 设备
- 可科学上网
- 具备良好英文水平

---

## 核心策略：从图形界面到命令行的渐进式学习

### 为什么不直接学 Claude Code CLI？

对于非技术人员：
- ❌ **命令行门槛高**：需要学习 Terminal、Shell 命令、Git 等
- ❌ **概念抽象**：文件路径、环境变量、进程管理等
- ❌ **错误难调试**：看不懂报错信息，不知道如何解决
- ❌ **缺乏即时反馈**：纯文本交互，缺少可视化

### 推荐路径：Cowork → Claude Code

**阶段 1：Claude Cowork（图形界面）**
- ✅ **零门槛**：点击安装，图形界面操作
- ✅ **可视化**：看得见 AI 在做什么（文件操作、任务执行）
- ✅ **安全沙箱**：隔离环境，不会破坏系统
- ✅ **即时反馈**：实时看到 AI 的推理过程和执行结果

**阶段 2：Claude Code CLI（命令行）**
- 在掌握 AI 协作模式后，逐步过渡到更强大的 CLI 工具
- 此时已理解 AI 的工作方式，学习命令行更有目标

## 方案对比：官方 vs 开源

### 方案 A：Claude Cowork（官方，推荐）

**优势：**
- ✅ **最佳体验**：Anthropic 官方产品，稳定可靠
- ✅ **最强模型**：Claude Opus 4.6，100万 token 上下文
- ✅ **完整功能**：文件操作、多任务协调、专业输出（PPTX、DOCX、XLSX）
- ✅ **安全隔离**：VM 级别隔离，数据本地化
- ✅ **持续更新**：官方支持，功能不断增强

**劣势：**
- ❌ **需要付费**：Pro $20/月 或 Max $100/月
- ❌ **需要科学上网**：访问 Anthropic 服务

**适合人群：**
- 预算充足（$20-100/月）
- 追求最佳体验和稳定性
- 可以科学上网

**安装步骤：**
```bash
# 1. 访问 https://claude.ai
# 2. 订阅 Claude Pro ($20/月) 或 Max ($100/月)
# 3. 下载 Claude Desktop for macOS
# 4. 启动 Cowork 功能
```

---

### 方案 B：Open-Claude-Cowork（开源，替代）

**优势：**
- ✅ **完全免费**：开源项目，无订阅费用
- ✅ **支持国内模型**：GLM、MiniMax、Kimi、DeepSeek
- ✅ **一键安装**：无需编程知识
- ✅ **灵活配置**：支持多种 API（Claude、OpenAI、国内模型）
- ✅ **本地化友好**：支持中文模型，无需科学上网（使用国内模型时）

**劣势：**
- ❌ **社区维护**：非官方项目，更新可能不及时
- ❌ **功能可能不完整**：某些高级功能可能缺失
- ❌ **需要自行配置 API**：需要获取 API Key

**适合人群：**
- 预算有限或希望零成本起步
- 无法科学上网或希望使用国内模型
- 愿意尝试开源方案

**安装步骤：**
```bash
# 1. 访问 https://github.com/OpenCoworkAI/open-cowork
# 2. 下载 macOS 安装包
# 3. 配置 API Key（可选择国内模型如 GLM、Kimi）
# 4. 启动应用
```

**推荐国内模型：**
- **智谱 GLM-4**：$0.1/1M tokens，中文能力强
- **Kimi**：长上下文（200K tokens），适合文档处理
- **DeepSeek Coder**：代码能力强，适合编程任务
- **MiniMax**：综合能力均衡

#### Prompt Engineering 技巧

**好的 Prompt 结构：**
```
[背景] 我是一名市场经理，需要分析客户反馈
[目标] 从 100 条反馈中提取关键问题
[约束] 按优先级排序，只保留前 10 个问题
[格式] 输出为 Markdown 表格，包含问题、频次、优先级
```

**常见错误：**
- ❌ 太模糊："帮我处理这些文件"
- ✅ 具体明确："将这 50 个 PDF 文件按日期重命名，格式为 YYYY-MM-DD_原文件名.pdf"

#### Day 18-21：MCP 工具集成

**什么是 MCP？**
- Model Context Protocol：让 AI 连接外部工具
- 例如：浏览器、Notion、数据库、API

**示例：连接浏览器**
```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-playwright"]
    }
  }
}
```

**实战任务：**

**对比 Cowork 和 CLI：**
| 特性 | Cowork（图形界面） | Claude Code CLI |

## 常见问题

### Q1: 我完全不懂编程，能学会吗？
**A**: 可以！Cowork 就是为非技术人员设计的。你只需要：
- 会用电脑（文件管理、浏览器）
- 会用自然语言描述任务
- 愿意尝试和学习

### Q2: Cowork 和 ChatGPT 有什么区别？
**A**:
- **ChatGPT**: 对话式 AI，需要你手动复制粘贴文件内容
- **Cowork**: 自主式 AI，可以直接读写你的文件，执行多步骤任务

### Q3: 数据安全吗？
**A**:
- **官方 Cowork**: VM 隔离，数据本地化，不用于训练
- **开源版本**: 完全本地运行，你控制 API 调用

### Q4: 需要多少预算？
**A**:
- **最低**: $0（开源版本 + 国内免费模型）
- **推荐**: $20/月（Claude Pro）
- **高级**: $100/月（Claude Max，重度使用）

### Q5: 学完后能做什么？
**A**:
- ✅ 自动化日常办公任务（文档处理、数据分析）
- ✅ 构建个人工作流（邮件管理、知识库）
- ✅ 过渡到编程学习（如果感兴趣）
- ✅ 提升工作效率 50%+
